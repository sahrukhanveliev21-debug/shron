# 🤖 Telegram Account Manager

Telegram бот для управления несколькими аккаунтами одновременно. Отслеживает новые сессии на твоих аккаунтах и позволяет их блокировать или одобрять.

## 🎯 Возможности

- ➕ **Добавление аккаунтов** - добавляй до 16+ аккаунтов
- 📋 **Список аккаунтов** - смотри статус каждого (онлайн/офлайн)
- ❌ **Удаление аккаунтов** - удаляй ненужные
- 🔄 **Рестарт** - перезапускай все сессии
- ⚠️ **Уведомления** - получай оповещения о новых входах
- 🚫 **Блокировка сессий** - кикай подозрительные входы
- ✅ **Одобрение сессий** - оставляй свои входы

## 📦 Установка

```bash
pip install -r requirements.txt
```

## ⚙️ Конфигурация

Отредактируй переменные в `main.py`:

```python
API_ID = 2040                              # Твой API ID от Telegram
API_HASH = 'b18441a1ff607e10a989891...'  # Твой API HASH
BOT_TOKEN = '8351363618:AAHmq7zZ...'     # Токен твоего бота
ADMIN_IDS = [5454585281]                  # ID администраторов
```

## 🚀 Запуск

```bash
python main.py
```

## 📌 Команды бота

- `/start` - главное меню

## 🎮 Кнопки

| Кнопка | Описание |
|--------|----------|
| ➕ ДОБАВИТЬ | Добавить новый аккаунт по номеру телефона |
| 📋 СПИСОК | Показать все добавленные аккаунты и их статус |
| ❌ УДАЛИТЬ | Удалить аккаунт из системы |
| 🔄 РЕСТАРТ | Перезапустить все сессии |

## 📊 Как это работает

1. **Добавление аккаунта**: Вводишь номер → вводишь код → вводишь пароль (если есть 2FA)
2. **Мониторинг**: Бот следит за новыми входами на каждый аккаунт
3. **Оповещение**: При новом входе бот отправляет инфо (устройство, IP, страна, время)
4. **Действие**: Ты выбираешь - кикать или оставить сессию

## 💾 База данных

Все аккаунты хранятся в `accounts.db` (SQLite).
Сессии сохраняются в папке `sessions/`.

## 🔐 Безопасность

- Только администраторы имеют доступ к командам
- Сессии шифруются Telethon
- Локальное хранилище баз данных
